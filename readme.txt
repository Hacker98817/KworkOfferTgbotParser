Телеграм бот который парсит сайт upwork по выбранным фильтрам в виде inline кнопок и отправляет результаты (Название, ссылка, дата публикации заказа, и другую информацию) . Данные о фильтрах хранятся в базе данных, парсинг происходит раз в час и отправляет только те заказы, которые не были отправлены ранее пользователю
(пользователь может добавлять группы или удалять их с помощью кнопок). Данные о заказах хранятся в кэше с помощью redis. 

Запуск с файла bot/telegram_adapter/main.py
